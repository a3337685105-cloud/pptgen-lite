# PPT Studio v2 设计规范 (Design Specifications)

## 1. 核心视觉目标
*   **专业感**：通过精准的间距和对齐，体现生产力工具的严谨。
*   **交互感知**：通过平滑的过渡动画，让 AI 生成的过程显得“丝滑”。
*   **内容优先**：减少装饰性元素，让用户的注意力集中在 PPT 内容和生成逻辑上。

## 2. 风格定义：极简玻璃拟态 (Minimalist Glassmorphism)
*   **背景**：浅灰色 (#F8F9FA) 或带有微弱极光效果的渐变背景。
*   **卡片**：半透明白色 (#FFFFFFBB)，带有 12px 的圆角和 `backdrop-filter: blur(20px)`。
*   **投影**：使用多层柔和阴影，增加深度感而非生硬的黑影。
*   **字体**：
    - 主体：Inter, -apple-system, "Segoe UI", "PingFang SC".
    - 标题：加粗，适当增加字间距。

## 3. 色彩系统
*   **Primary (品牌色)**：`#2563EB` (经典蓝) - 用于主按钮、激活状态。
*   **Success (成功)**：`#10B981` (翠绿) - 用于生成完成、Key 测试通过。
*   **Warning (风险)**：`#F59E0B` (琥珀色) - 用于 AI 拆分风险提示。
*   **Danger (错误)**：`#EF4444` (珊瑚红) - 用于报错、删除。
*   **Neutral (中性)**：`#64748B` - 用于辅助文字、次要边框。

## 4. 关键美化点
- [x] **侧边栏优化**：增加毛玻璃质感，优化导航项的 Hover 和 Active 动效。
- [x] **步骤条 (Ribbon Step)**：从生硬的矩形改为更具有指向性的连接设计，增加进度指示。
- [x] **卡片交互**：增加微小的浮起效果 (`transform: translateY(-2px)`)。
- [x] **按钮重塑**：从默认样式改为带有微弱渐变和内阴影的“实体感”按钮。
- [x] **输入框增强**：Focus 时增加呼吸灯式的 Glow 效果。
- [ ] **Canvas 容器**：增加深色背景切换，模拟 PPT 播放环境。

## 5. 动画原则
*   **持续时间**：标准动效 200ms，复杂布局 400ms。
*   **曲线**：使用 `cubic-bezier(0.4, 0, 0.2, 1)` 实现自然感。
