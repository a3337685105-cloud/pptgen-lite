# PPTGEN 使用指南

PPTGEN 是一个把文字内容生成 PPT 图片页，并导出为 PowerPoint 文件的桌面工具。当前版本生成的每一页本质上是一张图片，因此它适合快速制作视觉完整的演示稿；如果图片里有文字错误，可以重新生成、局部改图，或在 PowerPoint/WPS 里用色块覆盖后手动补字。

## 1. 安装与启动

1. 双击 `PPTGEN-1.0.0-x64.exe` 安装包。
2. 按提示选择安装目录。
3. 安装完成后，从桌面快捷方式或开始菜单打开 `PPTGEN`。
4. 首次启动可能需要几秒钟，因为软件会同时启动本地服务和桌面窗口。

API Key 会保存在本机应用数据里。关闭软件后再次打开，通常不需要重新填写。

## 2. 准备 API Key

进入软件后，先在设置或首页的 Key 区填写需要的密钥：

- `DashScope / Qwen API Key`：用于文本拆分、内容整理和部分提示词生成。
- `GPT Image 2 API Key`：用于 GPT Image 2 生图。
- `Grsai API Key`：用于 NanoBanana 2 / NanoBanana Pro 生图。

如果只使用 GPT Image 2，也仍建议填写 DashScope / Qwen Key，因为内容拆分阶段会用到文本模型。

## 3. 选择生图模型

打开软件后先选择最终生图模型：

- `GPT Image 2`：推荐优先使用，整体效果最好，适合高质量成稿。
- `NanoBanana 2`：速度和效果较均衡，适合批量生成和快速迭代。
- `NanoBanana Pro`：适合更复杂、更精细的画面尝试。

选择模型后点击确认，进入内容阶段。

## 4. 输入内容并拆分

在“内容与拆分”中输入你的原始文字，可以是大纲、讲稿、研究内容或已有 PPT 文案。

常用方式有两种：

- `拆分并扩写`：适合只有大纲或内容偏少的情况，系统会拆页并扩展到更适合上屏的字数。
- `仅拆分`：适合原文已经比较完整，只希望按页整理。

如果你使用 GPT 链路，也可以点击“跳过拆分”，直接进入手工生成模式，然后手动添加页面、填写标题和正文。

## 5. 逐页生成

进入“逐页确认与生成”后，界面分为三块：

- 左侧：页面列表。
- 中间：当前页标题、正文和风格关键词。
- 右侧：生图/改图区。

建议先检查每一页的标题和正文，再点击“生成该页”。如果你希望整组页面保持同一种视觉倾向，可以在“风格关键词”中写一句简短风格要求，例如：

```text
浅色、通透、蓝灰配色、学术期刊感
```

这个风格关键词会在页面之间保留，除非你手动修改。

## 6. 修改图片

如果生成结果不满意，可以使用三种方式调整：

1. `重新生成该页`：重新生成整张图片。
2. `按要求修改`：在“该页提示词”里写修改要求，再点击按钮。
3. 在画布上用画笔或矩形圈出位置，再写修改要求。系统会把手绘标记作为定位参考，并要求模型在完成修改后去掉手绘标记。

常见修改写法：

```text
把右侧图片换成更真实的实验室反应釜，整体颜色保持不变。
```

```text
把标题改得更醒目，但不要改变页面结构。
```

```text
修正图中公式错误，保持当前版式和配色。
```

## 7. 查看、保存与历史版本

每一页可以生成多个历史版本。点击底部缩略图可以切换版本。

可用操作：

- `查看大图`：打开当前图片的大图窗口。
- `另存图片`：保存当前页图片。
- `复制提示词`：复制实际发送给生图模型的完整提示词，方便到其他模型中测试。

## 8. 导出 PPT

所有页面确认后，点击“一键导出 PPT”。导出的 PPT 文件会保存在软件运行目录旁边的 `exports` 文件夹中。

注意：导出的 PPT 每页是图片，不是可直接编辑的文本框和图形对象。

## 9. 文字错误怎么处理

由于 AI 生图模型可能会出现文字渲染错误，可以按优先级尝试：

1. 重新生成该页。
2. 在改图区圈出错误文字，输入修改要求。
3. 如果只是少量文字错误，在 PowerPoint/WPS 中用相同颜色色块覆盖，再手动输入正确文字。

## 10. 常见问题

### 软件启动慢怎么办？

安装版会比单文件便携版更稳定。首次启动可能稍慢，后续通常会更快。

### API Key 会不会每次都要填？

不会。桌面版会把设置保存在本机应用数据中。除非清理应用数据、换电脑或换系统用户，否则通常会保留。

### 生成失败怎么办？

先检查：

- API Key 是否正确。
- 当前选择的模型是否和 Key 对应。
- 网络是否能访问模型服务。
- 提示词是否过长或包含模型不支持的内容。

如果单页失败，可以只重新生成这一页，不影响其他页面。

### 可以把这个工具发给别人用吗？

可以。推荐分发安装包 `PPTGEN-1.0.0-x64.exe`。对方安装后填写自己的 API Key 即可使用。
